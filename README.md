# Simple Calculator

## 📌 Project Overview
Simple Calculator is a Python-based application developed to perform basic arithmetic operations. This project provides an easy way for users to calculate mathematical expressions using a simple interface.

## ✨ Features
- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Fast and accurate calculations
- Easy-to-use program structure

## 🛠 Technologies Used
- Python 3

## 📋 Requirements
- Python 3.x

## 🚀 How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run the following command in the terminal:

```bash
python calculator.py
```
